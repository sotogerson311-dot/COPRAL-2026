La computación es la ciencia y el proceso de 
automatizar la gestión de información mediante el 
uso de ordenadores, combinando fundamentos teóricos
 (algoritmos) y prácticos (hardware y software). 
 Abarca el estudio, diseño, desarrollo e 
 implementación de sistemas para procesar, 
 almacenar y transmitir datos de manera eficiente y 
 segura.
